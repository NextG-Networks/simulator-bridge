| Key                     | Type / Example                 |  “Standard” starting value |     Typical range | What it does                                                                                                                |
| ----------------------- | ------------------------------ | -------------------------: | ----------------: | --------------------------------------------------------------------------------------------------------------------------- |
| `configuration`         | `int` (e.g., `0`)              |                        `0` | scenario-specific | Scenario layout/profile selector used by the C++ side.                                                                      |
| `bufferSize`            | `int` (project-specific units) |                       `10` |             5–100 | RLC/PDCP buffer sizing knob (per UE/flow depending on scenario). Larger helps bursty traffic at the cost of memory/latency. |
| `rlcAmEnabled`          | `0/1`                          |                        `1` |             {0,1} | **1** = RLC Acknowledged Mode (reliable, more overhead). **0** = UM (lower latency, possible loss).                         |
| `ues`                   | `int`                          |      `1` (smoke), `4` (RL) |              1–16 | Number of UEs.                                                                                                              |
| `n_gnbs`                | `int`                          |                        `1` |               1–2 | Number of gNBs. Handover logic needs ≥2 + mobility to actually trigger.                                                     |
| `e2nrEnabled`           | `0/1`                          |                        `1` |             {0,1} | Enable E2 / near-RT RIC integration & KPM indications.                                                                      |
| `simTime`               | `float` seconds                | `0.7` (smoke), `30.0` (RL) |             1–300 | Total simulation time. Must be ≥ `indicationPeriodicity`.                                                                   |
| `indicationPeriodicity` | `float` seconds                |                      `0.1` |          0.05–1.0 | Interval between E2 KPM reports; 100 ms is common.                                                                          |
| `hoSinrDifference`      | `float` dB                     |      `5` (smoke), `3` (RL) |               2–6 | SINR gap to prefer target cell for HO. Lower = more aggressive HO.                                                          |
| `dataRate`              | number/string (see notes)      |      `0` (smoke), `1` (RL) |              0–10 | Offered traffic per UE. **0 disables traffic** (control-plane only).                                                        |
| `useSemaphores`         | `0/1`                          |                        `1` |             {0,1} | Synchronize sim steps with the gym/agent. Set 0 only for unsupervised batch runs.                                           |
